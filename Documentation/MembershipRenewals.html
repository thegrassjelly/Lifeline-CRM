Membership Renewals


The membership renewal list or renewals list lets the customer view renewals submitted by the users, by default this is sorted to the newest ones at the top

like all of the other master list. Expanding the list lets the administrator view the specific user information which the renewal belongs to. The rightmost icon which looks like a folder will redirect the administrator to the specific renewal details as seen earlier.



The Deposit slip uploads list by default sorts and filters renewal which have new uploaded deposit slips by the users, which helps administrators to monitor newly uploaded deposit slips.



The Renewal Voids list filters out membership renewals which are voided out by the administrators.
